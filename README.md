# 🌦️ Weather Image Recognition using Deep Learning

## 📌 Overview
This project uses deep learning and computer vision to classify weather conditions from images.

## 🚀 Features
- Image classification (Sunny, Rainy, Cloudy, etc.)
- Deep Learning model (CNN)
- Image preprocessing & prediction

## 🛠️ Tech Stack
- Python
- TensorFlow / PyTorch
- OpenCV
- NumPy, Matplotlib
